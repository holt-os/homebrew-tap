# Holt Homebrew tap

Install [Holt](https://github.com/holt-os/holt), an open-source personal agent OS, with Homebrew. Node is installed for you as a dependency, so you do not need npm first.

```bash
brew install holt-os/tap/holt
```

Then:

```bash
holt init      # trust a folder, pick a brain, set up memory
holt chat      # start talking
```

Upgrade later with `brew upgrade holt`. To remove: `brew uninstall holt`.

Docs: https://productsdecoded.com/holt
